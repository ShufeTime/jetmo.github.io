该命令的一般格式为： echo [ -n ]

字符串其中选项n表示输出文字后不换行；字符串能加引号，也能不加引号。

用echo命令输出加引号的字符串时，将字符串原样输出；

用echo命令输出不加引号的字符串时，将字符串中的各个单词作为字符串输出，各字符串之间用一个空格分割。
